## Installation

You can have Gulp installed globally (not mandtatory): 

```sh
$ npm i -g gulp
```

```sh
$ npm install
```


## Compiling javascript and less:
### Continuous with watcher:

```sh
$ npm start
```

or

```sh
$ gulp
```

### just build once:
```sh
$ npm run compile
```
or 
```sh
$ gulp build
```

### build for production (minified javascript):
```sh
$ npm run compile:production
```
or 
```sh
$ gulp production
```
